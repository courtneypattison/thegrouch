## Install and run

First you need to create [Stripe API keys](https://stripe.com/docs/keys) and then add them to a .env file. See `settings.py` for which keys need to be created.

```
$ cd thegrouch
$ pip install -r requirements.txt
$ ./manage.py runserver
$ ../cd react-app
$ pnpm dev
```
